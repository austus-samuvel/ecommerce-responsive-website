# E-Commerce Responsive Website 🛒

A fully responsive e-commerce landing page built as part of the 
'Build Your Own Responsive Website' course on CCBP 4.0 NxtWave Academy.

## 🔗 Live Demo
ecommerceaus.ccbp.tech

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 4.5
- Font Awesome Icons

## ✨ Features
- Auto-sliding carousel banner (different images for mobile & desktop)
- Smooth-scroll fixed navbar linking to each section
- Product cards with interactive modals (Headphones & Laptops)
- Festive offers & deals section with responsive ordering
- Trending blogs section
- Explore deals & offers section
- Contact us section
- Fully responsive footer with social icons
- Tested across small, medium and large devices

## 📱 Responsive Breakpoints
| Device | Screen Size |
|--------|------------|
| Mobile | < 768px |
| Tablet | 768px - 991px |
| Desktop | ≥ 992px |
